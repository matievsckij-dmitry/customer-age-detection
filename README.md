# Определение возраста покупателей

**Описание**

Сетевой супермаркет «Хлеб-Соль» внедряет систему компьютерного зрения для обработки фотографий клиентов в кассовой зоне. Эта система позволит определять возраст покупателей, что будет полезно для анализа покупательских предпочтений и обеспечения соблюдения правил продажи алкогольной продукции.

**Цель**

Разработать модель, способную по фотографии предсказывать примерный возраст человека, используя предоставленный набор данных.

**Начальные данные**

Данные взяты с сайта ChaLearn Looking at People. Они находятся в папке /datasets/faces/. 
В распоряжении одна папка со всеми изображениями (/final_files) и CSV-файл labels.csv с двумя колонками: file_name и real_age. 

**Вывод**

В рамках этого проекта была успешно решена задача регрессии: по фотографии человека был определён его возраст. Для её реализации была разработана нейронная сеть на базе ResNet, которая показала результат MAE равный 5.9786. Также стоит отметить, что обучение модели проводилось достаточно быстро с использованием GPU, что открывает возможности для дальнейшего улучшения её качества. Например, можно применение различных схем аугментации данных для выбора наилучших вариантов, изменить число эпох обучения или поэкспериментировать с другими функциями активации, такими как Leaky ReLU или ELU.
